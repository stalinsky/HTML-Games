# HTML-Games

A collection of interactive games built with HTML, CSS, and JavaScript. This repository contains various browser-based games for entertainment and educational purposes.

## 📋 Overview

HTML-Games is a project featuring classic and modern games implemented using web technologies. Each game is self-contained and can run directly in any modern web browser without additional dependencies.

## 🎮 Features

- **Browser-based** - Play directly in your web browser
- **Responsive Design** - Works on desktop and mobile devices
- **Pure Web Technologies** - Built with HTML, CSS, and JavaScript
- **Easy to Play** - Intuitive controls and gameplay mechanics
- **Open Source** - Feel free to fork, modify, and improve

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/stalinsky/HTML-Games.git
   ```

2. Navigate to the project directory:
   ```bash
   cd HTML-Games
   ```

3. Open any game file in your web browser:
   - Simply double-click an HTML file, or
   - Use a local server for better compatibility

## 💡 Usage with Local Server

For optimal experience, run a local server:

```bash
# Using Python 3
python -m http.server 8000

# Using Python 2
python -m SimpleHTTPServer 8000

# Using Node.js (http-server)
npx http-server
```

Then visit `http://localhost:8000` in your browser.
